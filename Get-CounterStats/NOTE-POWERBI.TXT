To fix case :
- open the template (error will occur)
- Rename fields in the "FIELDS   >" section on far right
- Go to "Edit Queries" menu and load new CSV with case changed fields frin : Source gear icon under "APPLIED STEPS"
- Remove "Changed Type" 
- Highlight "Source" on the "FIELDS   >" section on far right
- If no data, then load your Sample file from the "Source " gear
- Highlight "Promoted Headers" and then menu 'Transform" 
            - select all table (CTRL + A)
            - click on the "Any column" section, "Detect Data Type" button

Click back on "Home" tab, "Close and Apply"
